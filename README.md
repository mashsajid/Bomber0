
<h1>Bomber0</h1>

<h5>Working mail bomber made with python</h5>

# How to config the bomber

```
python Bomber.py -c do
```
# Check Config

```
python Bomber.py -c check
```
# Send Mails

```
python Bomber.py -t [Recipient's Mail] -n [Number of Mails] -m [Message]
```
# Help

```
python Bomber.py -h
```
